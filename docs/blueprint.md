# **App Name**: Nexus

## Core Features:

- Dynamic Entrance Loader: A high-impact splash screen that displays the developer's name with a typography-focused animation before transitioning into the main site.
- AI Professional Tailor: An AI-powered tool that summarizes the developer's technical professional summary to match the vibe or industry of the visitor.
- Project Vault: Interactive grid displaying engineering projects with tech-stack tagging, role descriptions, and outcome highlights.
- Journey Timeline: A vertically scrolling experience map documenting internships and roles with time markers and expandable detail panels.
- Hero Contact Matrix: A centralized dashboard section in the hero including professional headshot and immediate links to GitHub, Instagram, Gmail, and LinkedIn.
- Seamless Section Anchor Navigation: A persistent header and footer system allowing fluid, animated jumps between bio, projects, and career milestones.

## Style Guidelines:

- Primary color: Electric Indigo (#7C3AED), a vibrant and tech-focused hue designed to pop against dark surfaces.
- Background: Deep Obsidian (#0D0C12), a heavily desaturated blue-black that creates an immersive, premium depth.
- Accent: Arctic Blue (#60A5FA), a cooler analogous contrast for secondary highlights and CTA elements.
- Headline font: 'Space Grotesk' (sans-serif) for a precise, engineering-forward aesthetic; Body font: 'Inter' (sans-serif) for readability.
- Code font: 'Source Code Pro' (monospace) for displaying tech stacks and implementation details.
- Minimalist grid-based layout with generous white space (negative space) and focused content density to create an 'expensive' agency feel.
- Smooth scroll-triggered transitions and micro-interactions on cards, utilizing a dark-mode blurring glassmorphism for floating UI elements.
